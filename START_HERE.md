## ⚡ Super Quick Start (5 Minutes)

### Terminal 1: Frontend
```bash
cd Frontend
npm install
npm run dev
```

### Terminal 2: Backend
```bash
cd Backend
npm install
npm run dev
```

### Terminal 3: Database
```bash
mysql -u root -p
source Backend/database.sql;
```

### Open Browser
```
http://localhost:5173
```

## 📝 Quick Reference

### Folders
- `Frontend/` - React app
- `Backend/` - Node.js API
- Docs at root level

### Start Commands
```bash
cd Frontend && npm install && npm run dev
cd Backend && npm install && npm run dev
```

### Access Points
- Frontend: http://localhost:5173

- Database: localhost:3306

### Key Files
- Animations: `Frontend/src/utils/animations.js`
- API: `Frontend/src/utils/api.js`
- Controllers: `Backend/src/controllers/`
- Routes: `Backend/src/routes/`
